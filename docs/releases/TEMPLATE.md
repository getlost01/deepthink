### Install

**Homebrew (recommended):**
```bash
brew tap getlost01/deepthink
brew update
brew upgrade --cask deepthink
```

**Manual:**
Download `DeepThink-X.Y.Z.zip`, unzip to `/Applications`, then:
```bash
xattr -rd com.apple.quarantine /Applications/DeepThink.app
```
