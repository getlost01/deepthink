## DeepThink 1.0.1

Stability and polish release following v1.0.0.

### App
- Boot splash loader during startup
- Improved service initialization and error handling across core services
- UI/UX polish: toast notifications, command palette fixes, trimmed built-in presets
- Release build fixes (Swift 6 Sendable, ad-hoc signing to prevent macOS "damaged" errors)

### CLI & MCP
- Rebuilt `deepthink` CLI and `deepthink-mcp` binaries (bundled in app resources)
- Version bumped to 1.0.1

### Distribution
- Homebrew cask updates automatically on release
- Quarantine cleared via cask postflight on install

### Install

**Homebrew (recommended):**
```bash
brew tap getlost01/deepthink
brew upgrade --cask deepthink
```

**Manual:**
Download `DeepThink-1.0.1.zip`, unzip, drag to `/Applications`, then:
```bash
xattr -rd com.apple.quarantine /Applications/DeepThink.app
```
