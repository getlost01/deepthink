## DeepThink 1.0.2

Theme support and design-system polish.

### App
- **Appearance:** System, Light, and Dark themes in Settings → General → Theme
- Theme syncs across app chrome, markdown editor, AI chat markdown, terminal, and Quick Capture
- Refined semantic color palette via `DSThemeManager`
- Quick Capture sheet modal and editor chrome fixes
- Launch stability improvements

### CLI & MCP
- Rebuilt `deepthink` CLI and `deepthink-mcp` binaries (bundled in app resources)
- Version bumped to 1.0.2

### Distribution
- Homebrew cask updates automatically on release
- Quarantine cleared via cask postflight on install

### Install

**Homebrew (recommended):**
```bash
brew tap getlost01/deepthink
brew update
brew upgrade --cask deepthink
```

**Manual:**
Download `DeepThink-1.0.2.zip`, unzip, drag to `/Applications`, then:
```bash
xattr -rd com.apple.quarantine /Applications/DeepThink.app
```
