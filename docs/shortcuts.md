# Keyboard Shortcuts

## Global

| Shortcut | Action |
|----------|--------|
| `Option+Space` | **Quick Capture** — floating panel from any app (may require Accessibility; see [Quick Capture](app/quick-capture.md)) |
| `Cmd+K` | Command palette |

## Navigation

| Shortcut | Section |
|----------|---------|
| `Cmd+0` | Recent |
| `Cmd+1` | Workspace |
| `Cmd+2` | Knowledge |
| `Cmd+3` | Context Graph |
| `Cmd+4` | AI Assistant |
| `Cmd+5` | Reminders |
| `Cmd+6` | Integration |
| `Cmd+7` | Terminal |

## Workspace

| Shortcut | Action |
|----------|--------|
| `Shift+Cmd+1` | Projects |
| `Shift+Cmd+2` | Notes |
| `Shift+Cmd+3` | Tasks |
| `Cmd+N` | New Note |
| `Cmd+T` | New Task |
| `Shift+Cmd+N` | New Project |
| `Shift+Cmd+R` | New Reminder |
