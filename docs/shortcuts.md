# Keyboard Shortcuts

## Global

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+D` | **Quick Capture** — floating panel from any app (requires Accessibility permission) |
| `Cmd+K` | Command Palette |

## Navigation

| Shortcut | Section |
|----------|---------|
| `Cmd+0` | Recent |
| `Cmd+1` | Workspace |
| `Cmd+2` | Knowledge |
| `Cmd+3` | AI Assistant |
| `Cmd+4` | Integrations |
| `Cmd+5` | Reminders |
| `Cmd+6` | Terminal |

## Workspace

| Shortcut | Action |
|----------|--------|
| `Shift+Cmd+1` | Projects |
| `Shift+Cmd+2` | Notes |
| `Shift+Cmd+3` | Tasks |
| `Cmd+N` | New Note |
| `Cmd+T` | New Task |
| `Shift+Cmd+N` | New Project |
| `Shift+Cmd+R` | New Reminder |
