# Keyboard Shortcuts

## App

| Shortcut | Action |
|----------|--------|
| `Cmd+K` | **Quick Search** (command palette) |
| `Option+Space` | Quick Search alias (app-only) |
| `Cmd+D` | **Daily Brief** — AI summary of today's tasks, reminders, and notes |

## Navigation

| Shortcut | Section |
|----------|---------|
| `Cmd+0` | Recent |
| `Cmd+1` | Workspace |
| `Cmd+2` | Knowledge |
| `Cmd+3` | Context Graph |
| `Cmd+4` | AI Assistant |
| `Cmd+5` | Reminders |
| `Cmd+6` | Integrations |
| `Cmd+7` | Terminal |
| `Cmd+,` | Settings |

## Workspace

| Shortcut | Action |
|----------|--------|
| `Shift+Cmd+1` | Projects |
| `Shift+Cmd+2` | Notes |
| `Shift+Cmd+3` | Tasks |
| `Cmd+N` | New Note |
| `Cmd+T` | New Task |
| `Shift+Cmd+N` | New Project |
| `Shift+Cmd+R` | New Reminder |
